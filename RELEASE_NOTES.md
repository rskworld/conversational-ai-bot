### Installation

```bash
# Clone the repository
git clone https://github.com/rskworld/conversational-ai-bot.git
cd conversational-ai-bot

# Install dependencies
pip install -r requirements.txt

# Download NLTK data
```

**CLI Mode:**

**Web Interface:**
